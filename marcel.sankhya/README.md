# Sankhya Relatórios Financeiros

Repositório com os relatórios automatizados desenvolvidos para o sistema TRP (Sankhya):

- GeraFinanceiro
- RecebeParcialPromissoria
- RenegociaFinanceiro

Cada módulo está separado por pasta e contém código responsável por gerar ou processar dados financeiros no sistema.

## Tecnologias
- Java
- SQL
- GitHub

## Como usar
Clone o repositório e siga as instruções dentro de cada pasta.
